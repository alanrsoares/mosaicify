# mosaicify
Transform your pictures into mosaics

Check out the demo: http://alanrsoares.github.io/mosaicify/

### Install npm dependencies
```
  npm install
```

### Install bower dependencies
```
  bower install
```

### start dev server
```
  npm start
```
